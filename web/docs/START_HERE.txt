🐛 TROUBLESHOOTING:

Problem: Dependencies won't install
→ rm -rf node_modules && npm install

Problem: App won't start
→ npm start -- --clear

Problem: Can't connect to device
→ Ensure same WiFi network
→ Try: npm start -- --tunnel
